Local Storage

La Local Storage API permite almacenar datos de manera persistente en el navegador del usuario, incluso después de cerrar o recargar la página.

  • localStorage.setItem(key, value): Guarda un valor con una clave específica en el almacenamiento local.
  • localStorage.getItem(key): Recupera el valor almacenado asociado a la clave proporcionada.
  • localStorage.removeItem(key): Elimina un elemento específico del almacenamiento local.
  • localStorage.clear(): Borra todos los datos almacenados en el almacenamiento local.

La FullScreen API permite cambiar el contenido de la página web a pantalla completa, proporcionando una experiencia más inmersiva.

  • Element.requestFullscreen(): Pone el elemento en pantalla completa.
  • document.exitFullscreen(): Sale del modo pantalla completa y vuelve al estado normal.

La Screen Wake Lock API permite evitar que la pantalla de un dispositivo se apague o atenúe automáticamente mientras se está ejecutando una tarea importante.

  • navigator.wakeLock.request('screen'): Solicita que la pantalla permanezca encendida.
  • wakeLock.release(): Libera el bloqueo de la pantalla, permitiendo que se apague o atenúe normalmente.
  • document.addEventListener("fullscreenchange", ... ): Detecta cambios en el estado de pantalla completa y puede usarse para gestionar el bloqueo de pantalla.
